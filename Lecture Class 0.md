# Basic Command Line on terminal
### 1. ดูตำแหน่งปัจจุบันที่อยู่ในระบบ
```
pwd
```
### 2. list ทุกๆ file/folder ที่อยู่ใน folder ปัจจุบัน 
``` 
ls
```
```
ls -l
```
```
ls -ltr #บอกรายละเอียดไฟล์อย่างละเอียด
```
### 3. Change directory 
```
cd #ถอยออกมาสุด[Home]
```
```
cd .. #ถอยกลับออกจากโฟลเดอร์ปัจจุบัน 1 ครั้ง
cd ../.. #ออกจากโฟลเดอร์ปัจจุบัน 2 ครั้ง
```
```
cd .xxx/yyy/zzz #เปลี่ยน directory แบบระบุปลายทาง [ไปโฟลเดอร์ที่ระบุใน path]
```
```
cd filename/ xxx #กรณีที่ชื่อไฟล์มี spacebar คั่น Ex. Class 4 ต้องพิมพ์ `cd Class/ 4`
```
### 4. Directory
```
dir #แสดงรายละเอียด File และ Directory ที่มีอยู่ใน Path ปัจจุบัน
```
